# Skill Registry

## Goal

Provide a local registry of STM32 skills that the workflow can match against AI recommendations.

## Current scope

- Maintain canonical local skill IDs
- Maintain aliases that may appear in AI output
- Record category, summary, and expected output targets

## First entries

- clock/RCC init
- GPIO configuration
- USART configuration
- SysTick timing
- Keil project integration
- ADC sampling
- DAC output
- PID control
